ORA-00917: missing comma  - например, не задан формат даты при вставке новых данных в таблицу 
(insert into RABOT values (1,'Ivanov',1,100000,TO_DATE('2013-03-14','YYYY-MM-DD'));)

PLS-00201:  identifier 'ИМЯ' must be declared
  Причина:  Попытка обратиться к объекту, который либо не был объявлен, либо лежит вне сферы видимости.
 Действие:  Проверьте написание имени и место его объявления. пользователь\пакет\хранимый объект в begin ... end;
