Интерфейс пакета

1. Спецификация содержит ОБЩИЕ объявления, которые видимы вашему приложению.

2. Тело содержит детали реализации и ЛИЧНЫЕ объявления, которые скрыты от вашего приложения.

Можно отлаживать, развивать или заменять тело пакета, не изменяя интерфейс к этому телу (т.е. спецификации пакета ) 




ПРИЛОЖЕНИЕ <-> Пакет (спецификация \ тело пакета ) -> База данных ( - \ - )
